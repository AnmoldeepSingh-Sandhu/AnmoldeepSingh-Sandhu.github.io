How to use

Once you enter the site there will be shop now button. If you click that it will scroll up the page to iphone13 or you can do that manually.

There is also iphone12 and iphone11 that I added there if you click on iphone12/iphone11 image or buy link it will take you to iphone12.html/iphone11.html page. Once you reach either page it will show Out of stock text and that's all there is nothing more in these pages.

When you click on iphone13 image or buy link it will take you to iphone13.php. Where you can see image, form for submitting color and storage for your phone, and some Specifications about phone. Once you select the color and storage and click on buy button, it will take you to buy.php page. On this page you will fill up another form to order a phone for you. You will enter your first name, last name, address, email, any comment. Then in payment details cardholder name, card number, expiry date. Once you fill this form correctly and press process transaction button, all the data of this form and previous form will be stored in database expect payment details, in case if someone enter his real card details. Also, validation is applied on both forms so you should enter valid data otherwise it will show you error messages. After the data is submitted into database buy.php page will show you the details you entered in the both forms. All details you see will be fetched from database. There is also cancel transaction and close button on buy.php page, if you click these button it will cancel your order and take you back to iphone13.php page.

Features

  1. On iphone13 page whatever color you will select, it's radio button will also show you same color.
  2. If you enter invalid data in any form, it will show error message that will glow.
  3. If you click shop now button on home page it will scroll up to iphone13.
  4. I make a image gallery on iphone13 page.
